# Github Pages

Github Pagesはgithub上にサイトを公開できる機能です。

## 導入手順

1. githubにpublicなリポジトリを作成します。
    ```
    nop892371/test-page
    ```
2. `git clone`でローカルにコピーします。
3. 何かファイルを作成しpushします。
    何かpushしないとGithub Pagesを公開できません。
4. リポジトリのsettingを開きます。
    1. GitHub PagesのセクションでSourceを設定します。
